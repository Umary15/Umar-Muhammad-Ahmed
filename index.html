Week 1: Introduction to the Web

Just like every other journey we started with understanding the fundamentals of what the web is. The web, or World Wide Web, is a system of interlinked documents and resources accessed via the internet. I learned that the internet is a global network connecting millions of computers, enabling them to communicate and share useful information.

Web development is the process of building and maintaining websites. It involves both the visible part of a website (front-end) and the behind-the-scenes logic and databases (back-end). I also explored how networks allow devices to connect and exchange data, making the internet possible.

Websites work by using web servers to store content and web browsers to display it. When you enter a website address, your browser requests information from a server. The server then sends the requested data back to your browser, which renders the web page for you to view and interact with.

Getting Started with HTML

At this point, I honestly lost count of the weeks, but my journey continued as I delved into HTML. Learning HTML (HyperText Markup Language) was my first real step into building actual web pages. It was exciting to see how simple tags could structure content and bring a blank page to life.

I discovered how to use headings, paragraphs, links, images, and lists to organize information on a web page. Understanding the basics of HTML gave me the confidence to experiment and create my own simple websites. Each new tag I learned felt like unlocking a new superpower in web development.

This phase taught me that web development is a continuous learning process, and it's okay to lose track of time as long as you keep moving forward. The more I practiced, the more comfortable I became with building and structuring web content from scratch.
